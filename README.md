# 📖 ConquisBiblia - Año Bíblico Conquistadores - Vanguard Elite

¡Bienvenido al proyecto oficial del **Año Bíblico para Conquistadores**! Esta es una aplicación web y móvil moderna, diseñada específicamente para ayudar a los Conquistadores de la Iglesia Adventista del Séptimo Día a mantener un registro constante y motivador de su lectura bíblica diaria.

![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat&logo=vite&logoColor=white)

---

## ✨ Características Principales

- **🛡️ Diseño Vanguard Elite**: Una interfaz premium, oscura y moderna con estética de "vidrio" (glassmorphism) y animaciones fluidas.
- **📅 Seguimiento de 52 Semanas**: Dashboard interactivo para visualizar el progreso anual.
- **🏆 Sistema de Logros**: Gana medallas y alcanza niveles basados en personajes bíblicos:
  - **Nivel Daniel**: Consistencia y sabiduría.
  - **Nivel David**: Valentía en la lectura.
  - **Nivel Esther**: Compromiso y propósito.
- **📖 Lector Bíblico Integrado**: Acceso directo a las lecturas diarias (Reina Valera 1960).
- **📱 Mobile-First**: Optimizada para dispositivos móviles y lista para ser empaquetada con Capacitor.

---

## 🚀 Tecnologías Utilizadas

- **Frontend**: [React.js](https://reactjs.org/) con [Vite](https://vitejs.dev/) para una velocidad ultra rápida.
- **Estilos**: CSS3 nativo con variables dinámicas y diseño responsivo.
- **Iconos**: SVG optimizados y personalizados.
- **Despliegue**: Preparado para GitHub Pages / Vercel / Netlify.

## 🛠️ Instalación y Configuración

Sigue estos pasos para ejecutar el proyecto localmente:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/ano-biblico-conquistadores.git
   ```

2. **Entrar al directorio:**
   ```bash
   cd ano-biblico-conquistadores
   ```

3. **Instalar dependencias:**
   ```bash
   npm install
   ```

4. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```
